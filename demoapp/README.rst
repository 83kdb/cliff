Running demoapp
===============

Setup
-----

First, you need to create a virtual environment and switch to it::

 $ pip install virtualenv
 $ virtualenv .venv
 $ . .venv/bin/activate

Now, install the demo into the virtual environment::

 $ python setup.py install
